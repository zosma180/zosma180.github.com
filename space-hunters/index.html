About the game

A sci-fi adventure where you will live the experience of a space engineer who lost his sight during a mission.

This is an audiogame. So, no video for you. An accurate sound design will take you to a new world.

Whether you are a sighted player or not, are you ready for this challenge ?

Game features

  • Story-driven project
  • AI generated voices
  • Exploration
  • Enemy battles
  • Weapons and equipment
  • Controller and keyboard supported

How to start playing

  1. Download the zip file
  2. Unzip the download file
  3. Execute the "Space Hunters.exe" file

Keyboard commands

Key Action
S or Enter Interact with map objects
Shift+Arrows Fire in the chosen direction
Shift+D Use radiator
W Use medikit
D Switch weapon
E Reload the gun
Q Detonate the explosive attached to the target
A+Up Listen to your current map position
A+Down Listen to the current map description
A+Left Listen to the remaining gun ammo
A+Right Listen to the current health information
A+E Listen to the current map objects
A+Shift Listen to the hunters description
A+Spacebar Listen to the weapons description
