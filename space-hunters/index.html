About the game

A sci-fi adventure where you experience the life of a space engineer who lost his sight during a mission.

This is an audio game, so there are no visuals for you. An immersive and accurate sound design will transport you to a new world.

Whether you are sighted or blind, are you ready for this challenge?

Game features

  • Story-driven experience
  • AI-generated voices
  • Exploration
  • Combat encounters
  • Weapons and equipment
  • Controller and keyboard support

How to start playing

  1. Download the zip file
  2. Unzip the download file
  3. Execute the "Space Hunters.exe" file

Keyboard commands

Key Action
Escape Hold to return to the main menu
Arrow Keys Move around the map
S or Enter Interact with objects on the map
Shift + Arrow Keys Fire in the selected direction
Shift + D Use the radiator
W Use a medikit
D Switch weapons
E Reload your weapon
Q Detonate the explosive attached to the target
A + Up Arrow Hear your current map position
A + Down Arrow Hear the current map description
A + Left Arrow Hear the remaining ammo count
A + Right Arrow Hear your current health status
A + E Hear the objects currently on the map
A + Shift Hear the hunters description
A + Spacebar Hear the weapons description
